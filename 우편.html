서울방배1동 우편취급국(02-581-2004)

    ★우편 종류 설명★
  1. 민원우편 : 빠른으로만 존재하고 가격은 5,980원이다. 딱보면 안다
  2. 내용증명 : 똑같은 종이 3묶음+빈봉투 한장이 한셋트이다, 보통 3,780원으로 빠른으로 보낼수도 있다.
  3. 등기 : 준등기(1,500원),일반등기(2,570원),빠른등기(3,070)가 있다. 보통 갈색 큰봉투에 담겨있으며 일반or빠른등기는 우측 상단에 따로 표기를 해주신다
    (표기가 따로 없는건 모두 준등기, 단 큰 갈색 봉투인데 채권번호and당담자가 없는건 "일반 우편(470원이다)" 으로 보내야 한다)
  4. 일반우편 : 맞다 당신이 생각하는 그게 일반우편이다 가격은 380원 이다
    ※만약 일반우편 우측 상단에 "등기" 라고 적혀있다면 이것은 등기로 보내야 하며 가격은 2,480이다
*기본적으로 보통 -> 빠른으로 업그레이드 비용은 500원 이다



●우편 별거없다. 지금 당신의 손에는 추심직 분들이 중구난방으로 던져놓은 우편들이 있을것이다 먼저 분류작업을 하자.

1.비싼우편순으로 배열해라(민원우편 -> 내용증명 -> 등기 -> 일반)
2.기본형식이 아닌 특수한것들을 맨 위로 올려라(빠른, 일반우편 봉투인데 등기로 보내야하는것, 준등기봉투인데 일반으로 보내야하는것 등)
3. 전산등록을 해라(회계 -> 전도집행금 등록/수정, 등기번호는 1로)
4. 예상 경비를 확인하고 돈을 챙겨서 우편을 보낸다
5. 영수증을 확인하고 등기번호를 입력하면서 금액을 확인+수정 한다(회계 -> 전도집행금등록현황)
6. 영수증은 A4용지로, 남은돈+오늘자 경비 사용 내역을 프린트 해서 경리에게 제출한다
(7.내용증명 우편이 있었다면 받아온 서류를 당담자에게 분배한다)
